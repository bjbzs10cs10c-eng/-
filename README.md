# 随机活动选择器 (Random Activity Selector)

一个简单实用的 Python 程序，帮助你在多个待办事项中随机选择一项，支持活动列表的动态增减和数据持久化。

## 🌟 功能特色

- ✅ **随机选择活动** - 从预设活动列表中随机挑选一项
- ⚠️ **防重复选择** - 确保本次选择与上一次不同
- 💾 **数据持久化** - 活动列表自动保存到本地 JSON 文件，重启程序后依然保留
- 📝 **动态管理** - 支持实时添加和删除活动项目
- 🎨 **彩色输出** - 选择结果以红色高亮显示（支持大部分终端）
- 🌐 **跨平台兼容** - Windows / macOS / Linux 通用

## 📋 预设活动列表

- 练习 vibe coding
- 看书
- 学英语
- 多邻国
- 练字
- 锻炼

> ⚡ 你可以在运行时随时添加或删除活动！
